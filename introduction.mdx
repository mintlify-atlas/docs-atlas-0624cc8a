OpenComic is a powerful, feature-rich desktop reader designed for comics, manga, and digital books. Built with Electron, it offers a beautiful reading experience with extensive customization options, support for virtually all comic formats, and advanced features like AI-powered image enhancement and cloud storage integration.

## What is OpenComic?

OpenComic is a cross-platform desktop application that transforms how you read digital comics and manga. Whether you're a casual reader or a dedicated collector, OpenComic provides all the tools you need for an optimal reading experience.

## Key features

OpenComic comes packed with features that make it stand out from other comic readers:

### Comprehensive format support

- **Image formats**: JPG, JP2, JXR, JXL, PNG, APNG, AVIF, HEIC, WEBP, GIF, SVG, BMP, ICO
- **Compressed archives**: RAR, ZIP, 7Z, TAR, LZH, ACE, CBR, CBZ, CBA, CB7, CBT
- **Documents**: PDF, EPUB (Alpha support)
- **Background music**: MP3, M4A, MP4, WEBM, WEBA, OGG, OPUS, WAV, FLAC

### Reading modes and customization

- Manga read mode (right-to-left)
- Webtoon read mode (vertical scrolling)
- Double page view for spread layouts
- Reading in scroll or slide navigation
- Floating magnifying glass for detailed viewing
- Multiple interpolation methods: lanczos3, lanczos2, mitchell, cubic, linear, nearest

### Image adjustments

- Brightness, saturation, contrast controls
- Sepia tone, negative, and color inversion
- Colorize black and white images
- AI-powered upscaling and artifact removal

### Library management

- Master folders for organizing collections
- Favorite labels and custom labels
- Bookmarks and automatic reading progress
- Continue reading from where you left off

### External integrations

- Tracking with AniList and MyAnimeList
- OPDS catalog support
- Server connections: smb://, ftp://, ftps://, scp://, sftp://, ssh://, s3://, webdav://, webdavs://

### Input methods

- Custom keyboard shortcuts
- Gamepad navigation
- Mouse and touch controls
- Custom tap zones for quick actions

## System requirements

OpenComic runs on most modern computers:

- **Windows**: Windows 7 or later (x64, ARM64)
- **macOS**: macOS 10.15 (Catalina) or later (Intel and Apple Silicon)
- **Linux**: Most modern distributions (x64, ARM64)

<Note>
  Minimum 4GB RAM recommended for smooth operation, especially when working with large comics or PDFs.
</Note>

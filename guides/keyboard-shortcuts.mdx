OpenComic offers extensive keyboard navigation with customizable shortcuts for all major functions. You can customize these shortcuts in the application settings.

## Navigation shortcuts

### Browse mode

<AccordionGroup>
  <Accordion title="Basic navigation">
    | Shortcut | Action |
    | --- | --- |
    | `F5` | Reload current view |
    | `Ctrl+F` | Open search |
    | `Ctrl+G` | Open search with filter |
    | `Mouse3` (Back button) | Go back |
    | `Mouse4` (Forward button) | Go forwards |
  </Accordion>

  <Accordion title="Comic navigation">
    | Shortcut | Action |
    | --- | --- |
    | `Ctrl+Up` | Previous chapter/comic |
    | `Ctrl+Left` | Previous chapter/comic |
    | `Ctrl+Down` | Next chapter/comic |
    | `Ctrl+Right` | Next chapter/comic |
  </Accordion>
</AccordionGroup>

## Reading mode shortcuts

### Page navigation

| Shortcut | Action |
| --- | --- |
| `Left Arrow` / `A` | Previous page (or next in manga mode) |
| `Right Arrow` / `D` | Next page (or previous in manga mode) |
| `Space` | Next page |
| `Up Arrow` | First page (or scroll up in scroll mode) |
| `Down Arrow` | Last page (or scroll down in scroll mode) |
| `Home` | Jump to first page |
| `End` | Jump to last page |
| `Ctrl+Up` / `Ctrl+Left` | Previous chapter |
| `Ctrl+Down` / `Ctrl+Right` | Next chapter |

<Note>
  In manga reading mode, left and right arrow keys are automatically inverted to match right-to-left reading direction.
</Note>

## Mouse shortcuts

### Tap zones

OpenComic divides the reading area into a 3x3 grid of tap zones. By default:

<AccordionGroup>
  <Accordion title="Left click zones">
    - **Left zones**: Previous page
    - **Center zones**: Reset zoom
    - **Right zones**: Next page
  </Accordion>

  <Accordion title="Right click zones">
    - **Left zones**: Next page
    - **Center zones**: Context menu
    - **Right zones**: Previous page
  </Accordion>

  <Accordion title="Middle click zones">
    - **All zones**: Reset zoom
  </Accordion>
</AccordionGroup>

<Note>
  Tap zones automatically invert in manga reading mode to match the reading direction. You can customize tap zones or disable them in Settings → Shortcuts → Tap Zones.
</Note>

### Mouse buttons

- **Mouse3** (Back button): Next page in reading mode, go back in browse mode
- **Mouse4** (Forward button): Previous page in reading mode, go forwards in browse mode
- **Middle click**: Reset zoom

## Customizing shortcuts

You can customize all keyboard shortcuts in the application:

<Steps>
  <Step title="Open settings">
    Click the settings icon or press the appropriate menu key
  </Step>
  <Step title="Navigate to shortcuts">
    Go to Settings → Shortcuts
  </Step>
  <Step title="Select action">
    Choose the action you want to customize
  </Step>
  <Step title="Record new shortcut">
    Press the desired key combination to assign it
  </Step>
  <Step title="Save changes">
    Your custom shortcuts are automatically saved
  </Step>
</Steps>

<Warning>
  Some shortcuts may conflict with system shortcuts. If a shortcut doesn't work, try choosing a different key combination.
</Warning>

## Restoring defaults

To restore all shortcuts to their default values:

1. Open Settings → Shortcuts
2. Click "Restore defaults" at the bottom of the shortcuts page
3. Confirm the action

This will reset all keyboard shortcuts and tap zones to their original configuration.

## Windows installation

### Standard installation (recommended)

<Steps>
  <Step title="Download the installer">
    Download the appropriate installer for your system:
    - **x64 (64-bit)**: [OpenComic.Setup.1.6.5.exe](https://github.com/ollm/OpenComic/releases/download/v1.6.5/OpenComic.Setup.1.6.5.exe)
    - **ARM64**: [OpenComic.Setup.1.6.5.arm64.exe](https://github.com/ollm/OpenComic/releases/download/v1.6.5/OpenComic.Setup.1.6.5.arm64.exe)
  </Step>
  
  <Step title="Run the installer">
    Double-click the downloaded file. You may need to allow the app to make changes to your device.
  </Step>
  
  <Step title="Follow the setup wizard">
    Choose your installation location and preferences. The installer will create desktop and Start menu shortcuts.
  </Step>
  
  <Step title="Launch OpenComic">
    After installation completes, launch OpenComic from your Start menu or desktop shortcut.
  </Step>
</Steps>

### Portable installation

Portable versions don't require installation and can run from a USB drive.

- [OpenComic.Folder.Portable.1.6.5.exe](https://github.com/ollm/OpenComic/releases/download/v1.6.5/OpenComic.Folder.Portable.1.6.5.exe) - Self-extracting archive

## macOS installation

### DMG installation (recommended)

<Steps>
  <Step title="Download the DMG file">
    Choose the correct version for your Mac:
    - **Intel Macs**: [OpenComic-1.6.5.dmg](https://github.com/ollm/OpenComic/releases/download/v1.6.5/OpenComic-1.6.5.dmg)
    - **Apple Silicon (M1/M2/M3)**: [OpenComic-1.6.5-arm64.dmg](https://github.com/ollm/OpenComic/releases/download/v1.6.5/OpenComic-1.6.5-arm64.dmg)
  </Step>
  
  <Step title="Open the DMG file">
    Double-click the downloaded DMG file to mount it.
  </Step>
  
  <Step title="Drag to Applications">
    Drag the OpenComic app icon to your Applications folder.
  </Step>
  
  <Step title="Launch OpenComic">
    Open OpenComic from your Applications folder or Launchpad.
    
    <Warning>
      On first launch, you may see a security warning. Right-click the app and select **Open** to bypass this.
    </Warning>
  </Step>
</Steps>

## Linux installation

### Debian/Ubuntu (.deb)

<Steps>
  <Step title="Download the package">
    - **x64**: [opencomic_1.6.5_amd64.deb](https://github.com/ollm/OpenComic/releases/download/v1.6.5/opencomic_1.6.5_amd64.deb)
    - **ARM64**: [opencomic_1.6.5_arm64.deb](https://github.com/ollm/OpenComic/releases/download/v1.6.5/opencomic_1.6.5_arm64.deb)
  </Step>
  
  <Step title="Install the package">
    ```bash
    sudo dpkg -i opencomic_1.6.5_amd64.deb
    sudo apt-get install -f  # Install dependencies if needed
    ```
  </Step>
  
  <Step title="Launch OpenComic">
    Find OpenComic in your application menu or run `opencomic` from the terminal.
  </Step>
</Steps>

### Fedora/RHEL/openSUSE (.rpm)

<Steps>
  <Step title="Download the package">
    - **x64**: [opencomic-1.6.5.x86_64.rpm](https://github.com/ollm/OpenComic/releases/download/v1.6.5/opencomic-1.6.5.x86_64.rpm)
    - **ARM64**: [opencomic-1.6.5.aarch64.rpm](https://github.com/ollm/OpenComic/releases/download/v1.6.5/opencomic-1.6.5.aarch64.rpm)
  </Step>
  
  <Step title="Install the package">
    ```bash
    # Fedora/RHEL
    sudo dnf install opencomic-1.6.5.x86_64.rpm
    
    # openSUSE
    sudo zypper install opencomic-1.6.5.x86_64.rpm
    ```
  </Step>
</Steps>

## Verifying your installation

After installation, verify OpenComic is working correctly:

<Steps>
  <Step title="Launch the application">
    Open OpenComic from your applications menu or desktop shortcut.
  </Step>
  
  <Step title="Check the version">
    Go to **Help** > **About** to verify you're running version 1.6.5 or later.
  </Step>
  
  <Step title="Test with sample content">
    OpenComic includes sample pages from Pepper & Carrot. Try opening these to verify everything works.
  </Step>
</Steps>

## Updating OpenComic

### Automatic updates

- **Microsoft Store** and **Mac App Store**: Updates automatically
- **Snap** and **Flatpak**: Updates through system package manager

### Manual updates

For manual installations:

1. Download the latest version from the [releases page](https://github.com/ollm/OpenComic/releases)
2. Install over your existing installation
3. Your settings and library will be preserved

## Uninstalling OpenComic

<Tabs>
  <Tab title="Windows">
    - **Installed version**: Use "Add or Remove Programs" in Windows Settings
    - **Portable version**: Simply delete the OpenComic folder
  </Tab>
  
  <Tab title="macOS">
    - **App Store version**: Delete from Applications folder
    - **DMG version**: Delete from Applications folder
    - To remove settings: Delete `~/Library/Application Support/OpenComic`
  </Tab>
  
  <Tab title="Linux">
    **Debian/Ubuntu**:
    ```bash
    sudo apt remove opencomic
    ```
    
    **Fedora/RHEL**:
    ```bash
    sudo dnf remove opencomic
    ```
    
    **Snap**:
    ```bash
    sudo snap remove opencomic
    ```
    
    **Flatpak**:
    ```bash
    flatpak uninstall app.opencomic.OpenComic
    ```
    
    **AppImage**: Simply delete the AppImage file
  </Tab>
</Tabs>

## Troubleshooting installation

<AccordionGroup>
  <Accordion title="Windows: 'Windows protected your PC' warning">
    This is normal for applications downloaded from the internet. Click **More info** and then **Run anyway**.
  </Accordion>
  
  <Accordion title="macOS: 'OpenComic cannot be opened' error">
    Right-click the app and select **Open**. On the next dialog, click **Open** again. This only needs to be done once.
  </Accordion>
  
  <Accordion title="Linux: Missing dependencies">
    Install required libraries:
    ```bash
    # Debian/Ubuntu
    sudo apt-get install libgtk-3-0 libnotify4 libnss3 libxss1 libxtst6 xdg-utils libatspi2.0-0 libuuid1
    ```
  </Accordion>
  
  <Accordion title="Application won't start">
    1. Make sure your system meets the minimum requirements
    2. Try running from terminal to see error messages
    3. Check the [troubleshooting guide](/guides/troubleshooting) for detailed solutions
    4. Report issues on [GitHub](https://github.com/ollm/OpenComic/issues)
  </Accordion>
</AccordionGroup>

We welcome contributions to the OpenComic project! Whether you're fixing bugs, adding new features, improving documentation, or helping with translations, your help is appreciated.

## Issues

We use [GitHub Issues](https://github.com/ollm/OpenComic/issues) to track bugs and feature requests for OpenComic.

### Bugs

If you come across any bugs — even minor ones — please report them.

<Note>
Before reporting a bug, please check the [existing issues](https://github.com/ollm/OpenComic/issues) to avoid duplicates.
</Note>

When submitting a bug report, please include as much relevant information as possible to help us reproduce and fix the problem quickly.

### Feature requests

We welcome suggestions for new features or improvements!

Before submitting a feature request, please search the [existing issues](https://github.com/ollm/OpenComic/issues) to see if someone else has already proposed it.

When creating a feature request, please include as much relevant information as possible, such as:
- A detailed description of the feature
- Any examples or use cases
- How it would improve the user experience

## Code contributions

We welcome contributions to the OpenComic code! Whether you're fixing bugs, adding new features, or improving documentation, your help is appreciated.

<Note>
Whenever possible, try to maintain the current coding style.
</Note>

### Bug fixes

<Steps>
  <Step title="Small fixes">
    For small fixes or minor corrections, feel free to open a pull request directly.
  </Step>
  
  <Step title="Large changes">
    For larger changes or if you're uncertain, please open an issue or a draft pull request first to discuss your approach.
  </Step>
</Steps>

### Feature implementations

For major changes or enhancements, please open an issue or a draft pull request first so we can review your contribution.

### Tracking site support

If you want to add support for a new tracking site, you can refer to the existing implementations or use the example as a starting point:

- [Anilist implementation](https://github.com/ollm/OpenComic/blob/master/scripts/tracking/anilist/anilist.js)
- [MyAnimeList implementation](https://github.com/ollm/OpenComic/blob/master/scripts/tracking/myanimelist/myanimelist.js)
- [Example implementation](https://github.com/ollm/OpenComic/blob/master/scripts/tracking/example/example.js)

Additionally, you'll need to update both files to include the new site's information:
- [`tracking-sites.js`](https://github.com/ollm/OpenComic/blob/master/scripts/tracking/tracking-sites.js)
- [`tracking-sites-keys.js`](https://github.com/ollm/OpenComic/blob/master/scripts/tracking/tracking-sites-keys.js)

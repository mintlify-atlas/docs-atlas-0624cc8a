If you want to see OpenComic in your language, please help us translate! OpenComic currently has translations into 22 languages.

## Creating a new translation

To create a new translation for OpenComic, follow these steps:

<Steps>
  <Step title="Get the empty template">
    Start by downloading the [empty.json](https://github.com/ollm/OpenComic/blob/master/languages/empty.json) file from the repository.
  </Step>
  
  <Step title="Compare with existing translations">
    Compare your empty file to other translations (preferably [English](https://github.com/ollm/OpenComic/blob/master/languages/en.json) or [Spanish](https://github.com/ollm/OpenComic/blob/master/languages/es.json)) using a diff tool:
    - [Meld](https://meld.app/)
    - [WinMerge](https://winmerge.org/)
    - [Mergely](https://editor.mergely.com/) (web-based)
    - Or any other method you prefer
  </Step>
  
  <Step title="Activate the language">
    Edit the [`languagesList.json`](https://github.com/ollm/OpenComic/blob/master/languages/languagesList.json) file and add `"active": true` to your language. If the language is not in the list, you can add it.
  </Step>
  
  <Step title="Submit your translation">
    You can send a pull request or open an issue with:
    - The new translated file
    - The updated `languagesList.json`
    
    <Note>
    It is not necessary that the translation is complete.
    </Note>
  </Step>
</Steps>

The list of translators appears in the `About OpenComic` window and in the TRANSLATE.md file.

## Updating an existing translation

To update an existing translation:

<Steps>
  <Step title="Get the current file">
    Download the current translated file from the [languages folder](https://github.com/ollm/OpenComic/tree/master/languages).
  </Step>
  
  <Step title="Find missing translations">
    Compare it to another translation (preferably [English](https://github.com/ollm/OpenComic/blob/master/languages/en.json) or [Spanish](https://github.com/ollm/OpenComic/blob/master/languages/es.json)) using a diff tool, or search for empty strings `""` in the translation file.
  </Step>
  
  <Step title="Add your name (optional)">
    If you consider it appropriate, you can add your data to the list of translators in the file. This data appears in the `About OpenComic` window and in the TRANSLATE.md file.
  </Step>
  
  <Step title="Submit your update">
    Send a pull request or open an issue with the updated translated file.
    
    <Note>
    It is not necessary that the translation is complete.
    </Note>
  </Step>
</Steps>
